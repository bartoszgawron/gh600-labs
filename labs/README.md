## How to Get Maximum Value

- Do the official Microsoft Learn module first
- Then immediately do the corresponding lab(s)
- Use your own `gh600-practice` repository for all exercises
- After each lab, spend 5–10 minutes writing what you learned in `notes/personal-reference.md`
