# Contributing to GH-600 Labs

Thank you for your interest in contributing to this repository!

## How to Contribute

1. **Fork** the repository and create a new branch from `main`.
2. Make your changes following the existing structure and conventions.
3. Open a **Pull Request** with a clear description of what you changed and why.
4. Wait for review before merging.

## Guidelines

- Keep lab content aligned with the official [GH-600 study guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/gh-600).
- Follow the existing file and folder naming conventions.
- Write clear, concise documentation that is easy for learners to follow.
- Do not push directly to the `main` branch.

## Reporting Issues

If you find an error, outdated information, or have a suggestion, please [open an issue](../../issues/new) describing the problem or improvement.

## Code of Conduct

Be respectful and constructive. This is a learning resource intended to help people prepare for the GH-600 exam.
